Vue d'ensemble

This project uses Git to manage its source code. Instructions on Git use can be found at https://git-scm.com/doc.

Accès Web

Ce qui suit est le lien online du dépôt de sources.

https://github.com/mybatis/mybatis-3

Accès de façon anonyme

The source can be checked out anonymously from Git with this command (See https://git-scm.com/docs/git-clone):

$ git clone --branch mybatis-3.5.18 ssh://git@github.com/mybatis/mybatis-3.git

Accès pour les dévelopeurs

Only project developers can access the Git tree via this method (See https://git-scm.com/docs/git-clone).

$ git clone --branch mybatis-3.5.18 ssh://git@github.com/mybatis/mybatis-3.git

Accès derrière un firewall

Référez-vous à la documentation du dépôt d'archive utilisé pour plus d'informations sur l'accès derrière un firewall.
